
# CRYPTO STORE

CRYPTO STORE is an E-commerce website project built using REACT. It has features like filters, products page, wishlist page and cart page.

## Tech Stack

REACT

## Features

- Home page
- Authentication pages
- Product page
- Wishlisht page
- Cart page
